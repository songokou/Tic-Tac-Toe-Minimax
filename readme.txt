Readme for homework assignment 4, Tic Tac Toe

This program is written in Java. What you need to do is either 
1. put it in a same package in Eclipse and hit “Run”, or 
2. use command line to compile:
	“javac *.java”
	“java MinMax”

You will be able to run the program by changing the String in main method of MinMax.java.
for example: 
The input format should be “b X b b O b b b b” and the output format is <1 3 4 6 7 9>.
